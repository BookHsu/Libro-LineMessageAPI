# Libro.LineMessageAPI.Language.en-US

English XML documentation language pack for:

- `Libro.LineMessageAPI`
- `Libro.LineMessageAPI.Extensions`

Install together with the main SDK packages:

```bash
dotnet add package Libro.LineMessageAPI
dotnet add package Libro.LineMessageAPI.Extensions
dotnet add package Libro.LineMessageAPI.Language.en-US
```
